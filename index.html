突然想明白的事情记录

8.13 / 上海 / 暴雨

今天早上突然想明白了一些事情，
之前一直很执着于买东西，
喜欢分析朋友在朋友圈里晒的东西，
但是现在却感觉越来越空虚，
感觉其实很多东西的价值都太浮于表面。
在经济范围内买适合自己的东西和自己喜欢的东西，
不要去追求或者让别人知道你买了什么来证明什么。
形考发现，相比这些，我觉得我还是更喜欢攒钱赚钱带来的满足感。
所以我今天做了后面的打算：

  • 省钱，尽可能地省钱来积攒积蓄；
  • 赚钱，赚钱的方法有很多，行动起来；
  • 学习理财，尝试多种理财方式（股票/外汇/黄金等）；
  • 精进技能，在学习中不断精进磨炼技能；
  • 增加资产或者能带来价值的东西，比如能持续带来收入的app/游戏/或者是建模。
    总之就是开源节流＋投资自己。
    默默努力，积极行动，闷声发大财。

吃瓜感想

8.10 / 上海 / 晴

上周六吃了婆婆精彩的瓜，现在一直在瓜田里面走不出来。
虽然是松哥之前推理的意料之中，但是还是惊讶于里面的故事，
这些事不好和同学朋友说，但是为了一过嘴瘾我要在这边说说，因为真的是在太爽又太气了~
在开始其实我一直认为她是一个很傻很天真的人，
但是知道订婚开始前，不知道为什么一直在后面说我和我家甚至是福建人的坏话，
开始只是不理解，但是到后面订婚完开始各种骚操作，又是诬陷我们家又是诋毁，还复刻当年她结婚的不爽也全留给我。
我的底线一次次被挑战，这让我非常气，那段时间导致我跟松哥经常吵架，甚至快要分开。
从那时候开始我一直觉得她并不傻并不天真，是个很会算计的人很会装的人。
到最近一直觉得越来越奇怪，我妈想在过年的时候寄一些年货给她，但是她却说等过完年寄到公司去，也不说家庭住址，谁问都不说。
她还跟松哥说为了省钱把其他房子租出去，自己租住在一个顶楼的自搭房里面，过得很辛苦，
而且周末从不视频，只打语音，三天两头对松哥发一次脾气，各种挑理发难，
明明每个月1万多的入账，还要向我们借钱周转，
说实话太多疑点了。
直到前天终于真相大白，
原来她早就婚内出轨，被发现后离婚，
甚至还搬到那个小9岁的男人家住在一起，那人还有一个小男孩，
而且这些年来那男人对她并不好，小孩也嫌弃她老说她不能当妈妈应该当奶奶，
这样说来一切都得到了解答，为什么钱不够用，为什么家庭住址不说，为什么总是对我们发脾气等。
但是更奇葩的是，她还特别绿茶，吃着碗里看着锅里，那边前夫要回老家，
就打电话钓着说复合，可是现在的感情又还没有切断，要前夫等到年底看看，
但是我觉得更奇怪的是为什么全部人都劝他们复合，
为什么不能在开始的时候就指出这样的错误，料想后面的结果呢？
所有人（特指李氏三姐妹及李母以及前夫家庭）的情感、三观及语言都很畸形，
作为一个局外人我觉得这样做有违人性，
作为一个会受到不良波及的局内人来说我觉得后面最难受的可能还是当事人，以及松哥和我，
如果复合我们可能面临的是听到她无止境的抱怨，无休止的干扰，和无边际的索要。
因为那些人就只顾自己，所有都只想到的是自己的利益，没人在乎其他人，这样一定很不开心，
但是畸形的价值观和感情的人们又怎么理解正常人的是非观和正常人的苦乐呢？

租房避坑

8.2 / 上海 / 晴

这次租房子给我们带来了深刻的教训，
整个人身体状况都不好了，而且最近这个感觉越来越严重了。
先来说说这个房子的缺点吧：

  • 设计不合理，导致经常受伤甚至缝针；
  • 湿度奇高，有时甚至高达97%，导致身体不能够散热特别是在夏天，皮肤病频发易拉肚子；
  • 房子没有窗户和阳光，造成了空气不流通，滋生霉菌是我和包子六一哮喘的直接原因；
  • 湿度大加上空气不流通，导致空气中氧气量不足，二氧化碳太多，给人疲倦感胸闷气短还引起了心肌缺血；
  • 虫子特别多，各种各样奇离古怪的虫子，见过的没见过，蚊子蟑螂蚂蚁就更不用说了；
  • 下水道设计不好，马桶排污直接连接浴室奇臭无比，还有喷粪阴影，厨房和阳台也是特别臭；
  • 之前卫生条件很差，在各种地方都是霉菌藏污纳垢，整理不干净；
  • 周围素质差，楼上经常性扔垃圾，衣服滴水至阳台，导致衣服不能放在外面晾；
  • 光线昏暗，白天如同晚上一般，在远离阳台3米以后的地方基本无光线，白天开灯；
  • 阳台一下大雨容易积水，下完容易沉淀泥土，需要经常性清理。
    最难受的还是身体状况每日愈下，造成整个人特别难受，最近租房价格又在上涨，也是让人越来越难受，可能因为这些种种，导致心情也不是很好，希望在下次租房的时候能吸取教训。

随便说说

8.2 / 上海 / 晴

上周五去尝试了一下自习室，感觉还不错，是很有学习氛围，特别安静，
但是主要的还是去到那边身体很多不适感竟然都得到了缓解，
所以从控制变量的角度来说，很大程度上，身体上的不适感或者说是生病，
大部分或者说全部原因都是由于现在的居住环境造成的。
因为觉得哮喘又犯了，所以在去自习室的前一天吃了两粒药，导致一直昏昏欲睡，但是学习效率竟然还不错。
本来希望是后面还可以去自习室的，也做好了这个准备，但是最近疫情有特别反复让我有点犹犹豫豫的。
昨天晚上聊天聊到太晚了，脑子一直很活跃，一直到天亮了都还没有睡着，今天状态也不是很好，
心脏有点疼，胸闷气短，甚至做肩胛骨后面也开始有点痛，很疲倦，手指还经常发麻，
买了一罐氧气吸了一下感觉有所缓解，但是还是有点不舒服，就是一直都是很不爽的状态。
希望能早点申请到公租房，摆脱这个糟糕的居住环境吧。
还有就是保持一个规律的作息，不能再熬夜了。
昨天晚上和松哥聊了很多八卦，感觉三观都被震碎了，只能说之前的感觉都是对的，以后还是少和没有道德底线的人接触，至少控制自己还是很容易的。
坚守自己的道德标准，但是己所不欲勿施于人，管好自己就好。

台风来啦

7.23 / 上海 / 晴

明天说是有强台风“烟花”要登来，
虽然从小到大每年都会见到听到台风已经司空见惯，
但是每次临近的时候还是会有一些紧张感，总想着要有所准备才心安。
然而，今天天气竟然出奇地好，太阳直射，闷热的空气像一层纱罩在皮肤上。
可能因为觉得提前完成了这周的减肥目标，所以这几天在吃东西上面就有点不受约束了，
今天早上不仅吃了两个桃子，一个蛋糕中午竟然还吃了一碗想了很久的酸辣粉，
所以为了弥补这次的方式，晚上就吃少一点，今天减肥操也多做一节好了。
新买的电脑和新买的T恤都好喜欢，就这些就可以让这个星期都很开心了。
今年租房的价格上涨不少，昨天看了这个小区甚至都没有房源了，
不知道明年合同到期了能否顺利住上公租房，或者是买到属于我们自己的公寓，但是一切都是值得被期待的。

随手记

2021.7.7 / 上海 / 晴

好久没有更新了，
最近发生了很多事情，
感慨很多，
生命的逝去，新生命的降生，生病带来的消极和不适，
无论哪一个事情单单挑出来都会有很多感慨，
感慨应该更加珍惜我爱的至亲们，
感慨时间过得太快，
感慨健康的重要。
身体最近也在慢慢恢复，作息也慢慢正常了，不会一直犯困，剧烈咳嗽到呼吸困难。
生活环境挺重要的，
这次租的房子也给了很大的教训。
不知道为什么最近总是有种着急和迫切想做什么事的感觉，
这种焦躁感让我每天有些不安，停不下来，
也可能是休息太久了吧，身体一恢复就有很多想做的事情。
因为不知道过敏性的哮喘到底是什么引起的，
所以最近添置了很多小家电——抽湿机、空气净化器还有空气循环扇，
没有很大的改变，但是又有一种明显的改变，说不清道不明。
一晃到了七月，愿至亲们后面都能平安顺遂，身体健康。
